# Docs

This directory is internal maintenance documentation for the package repo.

It is not part of the published `@aggc/ui` package and should only contain notes that help maintain:

- Storybook docs
- Playwright visual regression
- release and contribution workflows
- external consumption validation through `examples/vue-consumer/`

Primary executable tooling:

- `npm run docs`
- `npm run docs:build`
- `npm run docs:preview`
- `npm run test:visual`
- `npm run example:build`
